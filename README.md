# Camera WebApp

## Описание
Проект позволяет транслировать видео с нескольких камер в режиме реального времени через веб-интерфейс.

## Структура
- `main.py` - точка входа.
- `app/` - основной модуль:
  - `controller/` - обработка запросов.
  - `model/` - работа с камерами.
  - `view/` - HTML-шаблоны и статические файлы.
